# Fridge RNG Concept Summary

Core loop:

Roll Food/Clovers -> Feed Fridge -> Fridge Level rises -> Money/sec rises -> Buy upgrades/skilltree -> unlock better rarities -> Prestige -> rebuild faster.

Important rules:

- Fridge is a machine/object, not a pet. No personality, no hunger.
- Food is XP fuel for the Fridge.
- Money/sec depends on Fridge level, prestige, unique Mythic/Secret bonuses, upgrades and mutations.
- Common and Uncommon are available at start.
- Rare+ are unlocked through skilltree later.
- Clover Chains create the dopamine twist: a roll can become x2, x4, x8 ... up to a cap, then the final Food uses that luck.
- Mythic/Secret permanent bonuses should be unique-index based, not infinite duplicate stacking.
- Plots only, no zones for MVP.

MVP order:

1. Core game: registry, roll, inventory, plot, fridge XP/level, money/sec.
2. Progression: upgrades + skilltree.
3. Clover polish: chain UI, luck scaling, rare effects.
4. Long-term: index, prestige, mutations, merchant.
5. Polish: clean UI, sounds, leaderboards, daily rewards, monetization.
